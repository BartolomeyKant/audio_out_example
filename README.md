# AUDIO_OUTPUT_EXAMPLE

Пример кода для последовательного воспроизведения звуковых файлов на QT

Звуковые файлы находятся в `resources/`

## Сборка

Выполнить

```
cd audio_out_example
mkdir build
cd build
cmake ../
cmake --build .
```

## Запуск

После сборки выполнить

```
./audio_out_example
```
